# G-LAND v2.8.0 - 統合完成版

## 🎯 このリリースの位置づけ

**v2.7.19 / v2.7.20 の全機能 + v2.8.0 の Firebase 統合 + プラットフォーム別最適化を統合した完成版。**

これまでの全ての努力が結晶した「集大成」バージョンです。

---

## 🆕 統合された全機能一覧

### 【v2.7.19 由来】履歴機能の充実
- ✅ 履歴一覧の**コース/期間フィルタ**（5R/10R/All + コース選択）
- ✅ 履歴詳細の**「打数 / ±Par / 記号」3段階切替トグル**
- ✅ **記号表示ルール**：バーディ○、パーー、ボギー△、ダボ□、+3 以上は数値
- ✅ 同伴者表示改良（A案改良版：スコアなしでも参加者として記録）

### 【v2.7.20 由来】UX 改善多数
- ✅ **pendingJoin ゾンビ問題の修正**：ラウンド終了時にクリア
- ✅ **代理入力モーダル**：追加後も閉じずに継続入力可能
- ✅ **QRモーダル多層化解消**：招待表示が正しく動作
- ✅ **スコア画面のスクロール改善**：初回のみ自動、フローティングボタン追加
- ✅ **保存前確認ダイアログ**：全員入力完了後に確認
- ✅ **無料配布URL時の pendingJoin 自動クリア**
- ✅ **同伴者スコア同期改善**

### 【v2.8.0 新規】Firebase Authentication（Android/PC 用）
- ✅ **Google ワンタップログイン**（Android / PC）
- ✅ **Firebase UID による堅牢な識別**
- ✅ **機種変更後の自動データ復元**（Android）
- ✅ **email / photoURL の自動取得**

### 【v2.8.0 新規】プラットフォーム別最適化
- ✅ **iOS**：従来認証方式（姓かな）+ PWA フルスクリーン
- ✅ **Android**：Firebase Google ログイン + PWA フルスクリーン
- ✅ **共通**：PWA インストールバナー（ホーム画面上部）

### 【v2.8.0 新規】UX 抜本改善
- ✅ **全画面ローディング「くるくる」**
  - 主要な処理でスピナー表示
  - 3秒経過で「少しお待ちください」追加表示
  - 7〜10秒経過でキャンセルボタン
- ✅ **エラー UI の柔らかい表現**
  - 「エラー」より「もう一度お試しください」
- ✅ **緊急復旧機能**
  - マイページに「アプリをリセット」ボタン
  - localStorage / Cache / SW を全削除して初期状態に

### 【v2.8.0 新規】プロフィール2段階入力
- ✅ **初回登録：姓 + 姓かな（2項目のみ）**
  - スコア入力・同伴プレイヤー表示に必要な最小限
- ✅ **履歴閲覧時：名 + 名かな + ニックネーム 追加必須**
  - モーダルで入力を促進
- ✅ **マイページ**：5項目すべていつでも編集可能

### 【v2.8.0 新規】ニックネーム機能
- ✅ 旧「コース調整値」を廃止
- ✅ 代わりに「ニックネーム」項目を追加
- ✅ 表示ルール：
  - スコアカード：姓
  - ホーム/マイページ：ニックネーム（未設定時は姓）
  - リーダーボード（将来）：3択（姓 / ニックネーム / 匿名）

## 🚀 デプロイ手順

### 1. GAS 更新
1. GAS プロジェクトを開く
2. main.gs, api.gs, db.gs を上書き
3. デプロイの管理 → 編集 → 新バージョン → デプロイ

### 2. Users シート初期化（推奨）
既存テストデータをクリアするため、Users シート 2 行目以降を削除。
（firebaseUid / email / nickname 列は自動追加されます）

### 3. GitHub Pages デプロイ
```bash
git add .
git commit -m "Release v2.8.0: Integration of v2.7.19/v2.7.20 + Firebase Auth"
git push origin main
```

### 4. 動作確認
- Android Chrome: Firebase ログイン → 姓入力 → ホーム画面 → PWA インストール
- iPhone Safari: PWA Install Gate → ホーム画面追加 → 姓入力 → ホーム画面
- 両方: くるくるローディング表示、履歴閲覧時の追加項目プロンプト
